# For Customer — Vaha Ruchulu

Homemade Andhra pickles, podis & karams — customer-facing e-commerce website.

**Live site:** https://vaha-ruchulu.vercel.app

## Features

- Product catalog with cart & WhatsApp ordering
- Order tracking (Order ID + phone)
- Customer reviews
- Owner dashboard: orders, reviews, inventory (add/edit/hide items)

## Local development

```bash
npm install
npm run dev
```

For API routes locally, use `npx vercel dev`.

## Deploy

Connected to Vercel. Environment variables — see `.env.example`.
